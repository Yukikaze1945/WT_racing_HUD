这是一个基于 Python 的战争雷霆外置仪表盘，支持实时转速、档位显示和圈速计时。



###### 1.OBS 采集时请选择 "Windows 10 (1903+)" 模式并勾选 "允许透明度"。



###### 2.仅支持 Windows 系统。



手柄按键对照表

| 按钮代码 | Xbox 手柄对应 | PlayStation 手柄对应 |

| :--- | :--- | :--- |

| \*\*btn0\*\* | A | ❌ Cross |

| \*\*btn1\*\* | B | ⭕ Circle |

| \*\*btn2\*\* | X | 🟥 Square |

| \*\*btn3\*\* | Y | 🔺 Triangle |

| \*\*btn4\*\* | \*\*LB\*\* (左肩键) | L1 |

| \*\*btn5\*\* | \*\*RB\*\* (右肩键) | R1 |

| \*\*btn6\*\* | View / Back | Share / Select |

| \*\*btn7\*\* | Menu / Start | Options / Start |

| \*\*btn8\*\* | LS (左摇杆按下) | L3 |

| \*\*btn9\*\* | RS (右摇杆按下) | R3 |
